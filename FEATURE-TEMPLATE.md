### Title
Auto-populate birthdate when selecting user in patient form

### Description
When creating a patient record, selecting a user from the dropdown currently does not populate the *Date of Birth* field. This enhancement would automatically fill in the birthdate (and potentially other patient details) from the database once a user is selected.

### Steps to Reproduce
- Go to **Create Patient** form.
- Select a user from the dropdown (e.g., Juan Dela Cruz).
- Observe that the *Date of Birth* field remains empty.

### Expected Behavior
- After selecting a user, the system should automatically populate the *Date of Birth* field with the corresponding data from the database.
- Optional: Other patient details (e.g., contact info) could also be auto-filled for consistency.

### Benefits
- Reduces manual input errors.
- Speeds up patient record creation.
- Ensures consistency of patient data.
